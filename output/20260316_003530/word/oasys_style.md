# 機能仕様書

本文書は機能Aの仕様を定義する。

## 1. 概要

本システムはデータ管理を目的とする。

対象ユーザーは社内の業務担当者である。

## 2. 機能一覧

### 2.1 入力機能

CSV ファイルを読み込み、データベースに登録する。

入力ファイルの文字コードは UTF-8 とする。

### 2.2 出力機能

データベースから抽出した結果を帳票として出力する。

出力形式は PDF および Excel とする。

## 3. 非機能要件

### 3.1 性能要件

レスポンスタイムは3秒以内とする。

### 3.2 可用性要件

稼働率 99.9% 以上を目標とする。

## 4. 制約事項

オフライン環境で動作すること。

Windows 10 以降で動作すること。
