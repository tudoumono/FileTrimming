# .xls 取り扱いメモ

作成日: 260311 203032
更新日: 260311 203256

## 1. 結論

- `.xls` は Phase 2 で Excel COM により `.xlsx` へ正規化してから処理する
- 変換後は [`.xlsx 取り扱いメモ`](xlsx.md) と同じルートへ流す
- 原本は残し、品質問題が出たときに変換時点の崩れを追えるようにする

## 2. やり取り履歴

- `260311 203256`: 結論先行と履歴保持の形式へ更新した

## 4. 再確認しやすい論点

- 変換時に数式結果や書式が変わらないか
- 古い Excel 特有の埋め込みオブジェクトが欠落しないか
- 外部リンクや警告ダイアログを無人実行で抑止できるか

## 5. 試験時の確認項目

- `.xls` と `.xlsx` 変換後で表構造が保たれているか
- 数式セルの値が意図どおり再計算されているか
- 変換時の警告や保存ダイアログを抑止できるか
