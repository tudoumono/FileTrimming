## 1. 結論

- `.doc` は Phase 2 で Word COM により `.docx` へ正規化してから処理する
- 変換後は [`.docx 取り扱いメモ`](docx.md) と同じルートへ流す
- 原本は残し、変換起因の崩れを後追いできるようにする
